# Retail Lakehouse Pipeline

This project simulates a modern data engineering pipeline using Spark and Databricks.

## Goals
- Build a medallion architecture (bronze / silver / gold)
- Implement incremental data processing
- Handle real-world data issues (duplicates, late data, nulls)

## Tech Stack
- Spark (PySpark)
- Databricks
- Delta Lake
